truck-modeling : スイープとフェイス生成

このクレートはワイヤと曲面からシェル／ソリッドを構築します。主な演算は Extrude・Revolve・任意経路スイープです。

1. スイープ演算

  • Extrude : ワイヤを平行移動して側面とキャップを作成。エッジの生成元（generator / offset / rail）を記録します。
  • Revolve : 軸回りに回転。2π 超えや軸交差では面を分割して特異点を避けます。
  • Sweep-along-path : ガイド曲線に沿って断面を輸送。フレネ座標などで向きを制御します。

2. フェイスとシェル

  • ワイヤをサポートする曲面を選び、外ループ／内ループとしてトリムを定義します。
  • 生成された面を縫い合わせてシェルを構築。必要に応じて複数シェル（空洞付きソリッド）を作成します。
  • トリムは UV 空間で管理し、向き（外 CCW / 内 CW）を維持します。

3. チェックリスト

  • ワイヤが閉じており平面性が要件を満たしているか。
  • 選択したサーフェスが目的のスイープに適しているか（平面/ルールド/ロフト）。
  • エッジの派生関係を記録し、後続のブーリアンやフィレットで参照できるようにする。
  • トリムの UV ループが自己交差していないか確認する。
