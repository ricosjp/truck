truck-geometry : NURBS / ノットベクトル

truck-geometry は Truck の曲線・曲面実装を提供します。ノットベクトル、B-spline、NURBS の評価や コントロールポイント操作を行う際の注意事項を整理しました。

1. ノットベクトル

  • 非減少列で保持し、通常は 0–1 に正規化します。
  • 多重度が連続階数を決めるため、挿入・削除後は整合性を検証します。
  • 提供されるユーティリティで clamp / insert / remove を行い、直接書き換えは避けます。

2. 曲線評価

De Boor 型アルゴリズムで evaluate(t)、der(t)、der2(t) を計算します。 有効区間（例: [knots[p], knots[n-p])）を越えるパラメータは正規化後に評価します。 NURBS では重み付き同次座標で計算し、最後に正規化します。

3. 曲面

  • 二方向のノットベクトルとコントロールネットからテンソル積で生成。
  • uder / vder / uvder などの偏微分を備え、 ParametricSurface3D で法線を計算します。
  • トリム情報は truck-topology 側で管理されるため、UV 範囲を正確に返します。

4. コントロールポイントとリファイン

  • ローカル編集のみが影響するよう、ポイント移動後は該当スパンの再計算のみを行います。
  • 次数上げ、ノット挿入、再パラメトリ化のユーティリティが用意されています。
  • トレランスに依存する操作では truck-base の値を渡してください。
