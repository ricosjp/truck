概要

英語版ドキュメントの要点をかいつまんで日本語化しています。詳細や図版は英語版を参照してください。

  • 対象: Truck カーネル／ツール開発者
  • 更新: リポジトリの変更に追随
  • 構成: 幾何・トポロジ・モデリング・メッシュ・描画・STEP

モジュール一覧

  • truck-base — 幾何カーネルの基本規約
  • truck-geotrait — 曲線・曲面トレイト
  • truck-geometry — NURBS / ノットベクトル
  • truck-topology — B-Rep トポロジ
  • truck-modeling — スイープとソリッド生成
  • truck-shapeops — ブーリアン / ヒーリング
  • truck-meshalgo — テッセレーション / メッシュ処理
  • truck-polymesh — ハーフエッジメッシュ
  • truck-platform — WGPU プラットフォーム
  • truck-rendimpl — CAD 向けレンダラー
  • truck-derivers — 派生マクロ
  • truck-stepio — STEP I/O
