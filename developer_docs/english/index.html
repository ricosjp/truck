Overview

Truck is a modular CAD kernel. Each crate has its own quick-reference page in this directory; use them to align on coordinate conventions, APIs, and contributor expectations before opening PRs.

  • Updated: Continuous alongside the repo.
  • Audience: Kernel and tooling engineers.
  • Stack: Rust, cgmath64, WGPU, STEP.

Module guides

  • truck-base — coordinate system, tolerance strategy, and transform conventions.
  • truck-geotrait — parametric curve/surface traits and search helpers.
  • truck-geometry — knot vectors, B-spline/NURBS curves, surfaces, and refinement.
  • truck-topology — B-rep data structures and orientation rules.
  • truck-modeling — sweeps, shells, and solid builders.
  • truck-shapeops — boolean operations, healing, and diagnostics.
  • truck-meshalgo — tessellation, quadrangulation, and smoothing.
  • truck-polymesh — the shared half-edge mesh structure.
  • truck-platform — WGPU runtime, scene graph, and WGSL sandbox.
  • truck-rendimpl — CAD-specific renderer built on truck-platform.
  • truck-derivers — procedural macros for geometry traits and STEP derives.
  • truck-stepio — STEP import/export mappings.
