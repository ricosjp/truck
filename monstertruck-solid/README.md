# monstertruck-solid

[![Crates.io](https://img.shields.io/crates/v/monstertruck-solid.svg)](https://crates.io/crates/monstertruck-solid) [![Docs.rs](https://docs.rs/monstertruck-solid/badge.svg)](https://docs.rs/monstertruck-solid)

Crate for shape operations. Provides boolean operations on solids and shape healing for importing shapes from other CAD systems.

## Sample Codes

### punched-cube-shapeops

A punched cube generated by shape operator `AND`.
